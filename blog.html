Answer: The scope of variables depends on whether they are declared using var, let, or const.
  • var: Function-scoped, and variables are hoisted.
  • let: Block-scoped, and variables are not hoisted.
  • const: Block-scoped, similar to let, but cannot be reassigned.

Answer: Null and undefined are special values in JavaScript used to represent the absence of a value.
  • null: Intentional absence of any object value.
  • undefined: Default value of variables that are not initialized.

Answer: REST (Representational State Transfer) is an architectural style for designing networked applications.
  • An interface that follows the principles and constraints of REST
  • A set of rules and conventions for building and interacting with web services
  • Uses standard HTTP methods (GET, POST, PUT, DELETE).
  • Stateless communication between client and server.
  • Resources are identified by URIs (Uniform Resource Identifiers).
